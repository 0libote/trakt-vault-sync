# Trakt Vault Sync

An English-only Obsidian plugin that syncs Trakt watchlists, history,
favorites, and ratings into Markdown notes.

## Fork

Trakt Vault Sync is a personal, English-only fork of
[o1xhack/obsidian-sync-trakt](https://github.com/o1xhack/obsidian-sync-trakt).
That project was itself based on
[sarimabbas/traktr](https://github.com/sarimabbas/traktr).

## Development

```bash
npm ci
npm run dev
npm run build
npm run lint
```

`main.js` is generated and is not committed.
